I added a Bootstrap navbar at the top with the site name and links like “Browse Books” and “Cart.” 
On big screens, the links stay in one row, but on small screens, they collapse into a hamburger menu. Clicking the menu button 
shows or hides the links using Bootstrap’s collapse feature, which works with bootstrap.bundle.min.js. I also added Bootstrap 
spinners—small rotating circles—to show when data is loading, usually with a “Loading…” message so users know to wait.
